#!/usr/bin/env -S bunx --bun -p https://github.com/subtleGradient/gists/archive/refs/heads/x.tar.gz s example1 shebanger

example1.app.html

This is a static previewable document with a first-line Bun shebang.

The shebang uses bunx --bun to download the minimal x branch tarball and run shebanger.tsx as the package bin.

The stylesheet is a normal static link to shebanger.css through jsDelivr, so Spotlight can style the preview without JavaScript.

In a browser, shebanger.mjs loads as a normal ES module and adds a browser-mode diagnostics panel.
